# DeepStudy - 您的递归式深度学习助手

> DeepStudy 是一个基于 ModelScope 生态构建的智能学习工具，它将线性的 AI 对话转化为可视化的知识图谱，通过“递归式追问”帮助用户从零构建完整的知识体系。

---

## 1. 产品背景：为什么需要 DeepStudy？

在学习新概念或探索陌生领域时，我们常面临这样的困境：
*   **碎片化**：传统的 ChatGPT 类对话是线性的，知识点散落在漫长的聊天记录中，难以形成系统。
*   **浅尝辄止**：遇到不懂的术语，往往需要通过新开对话去搜索，打断了思维的连续性。
*   **缺乏脉络**：很难直观地看到知识点之间的逻辑联系和层级关系。

**DeepStudy** 旨在解决这些问题。它不仅仅是一个回答问题的机器人，更是一个帮你**画出知识地图**的助手。

---

## 2. 核心亮点

### 🌟 亮点一：思维导图实时生成 (Visual Knowledge Graph)
拒绝枯燥的纯文本输出。DeepStudy 会分析每一次问答的上下文，实时生成动态的思维导图。
*   **所见即所得**：左侧对话，右侧成图。
*   **结构化认知**：自动提取核心概念，将复杂的逻辑拆解为父子节点，帮助你一目了然地掌握知识脉络。

### 🚀 亮点二：划词递归追问 (Recursive Inquiry)
这是 DeepStudy 最独特的功能。我们模拟了人类深究问题的本能——“打破砂锅问到底”。
*   **交互创新**：在 AI 回复中看到任何感兴趣或不懂的术语，**直接用鼠标选中**。
*   **即时延伸**：系统会自动以选中的词为核心发起新一轮深挖，并在右侧思维导图中生成新的分支节点。
*   **思维不断流**：无需手动输入重复的提示词，学习过程如流水般顺畅。

## 3. 功能概览

| 功能模块 | 描述 | 用户价值 |
| :--- | :--- | :--- |
| **智能深度问答** | 基于 Qwen3-32B 强力驱动的对话引擎 | 无论是代码解释还是概念剖析，回答精准且富有逻辑。 |
| **动态知识树** | 随对话进程自动生长的节点树 | 将隐性的思维过程显性化，便于复习和回顾。 |

| **多轮对话管理** | 自动保存和管理历史会话 | 随时可以回到之前的某个知识分支继续探索。 |

## 4. 使用场景案例

### 场景 A：计算机系学生学习 "Transformer 架构"
1.  **初始提问**：用户问“什么是 Transformer 模型？”
2.  **生成全景**：AI 解释概念，右侧生成包含“Encoder”、“Decoder”、“Attention”等分支的导图。
3.  **递归深挖**：用户在回复中选中“Self-Attention”一词。
4.  **拓展分支**：AI 详细解释自注意力机制，导图中“Attention”节点下自动生长出“Q/K/V 矩阵”、“缩放点积”等子节点。
5.  **结果**：10分钟内，从一个概念拓展出一张完整的技术原理图谱。

### 场景 B：职场新人快速了解 "行业黑话"
1.  **阅读材料**：将一段晦涩的行业报告发给 DeepStudy。
2.  **划词解惑**：直接在 AI 的总结中划选不懂的术语（如“GMV”、“通过率”）。
3.  **建立联系**：通过导图看清这些指标之间的计算关系和业务逻辑。

## 5. 技术方案特色 (Powered by ModelScope)

虽然用户无需关心底层代码，但 DeepStudy 的流畅体验离不开以下技术支撑：

*   **模型底座**：深度集成 **Qwen** 系列模型，兼顾了通识理解与逻辑推理能力。
*   **图数据库驱动**：使用 **Neo4j** 存储对话结构，而非简单的文本存储。这使得系统能够真正“理解”知识点之间的层级关系。
*   **结构化提示工程**：精心设计的 Prompt Chain，确保 AI 输出的内容不仅准确，而且适合转化为结构化数据。
